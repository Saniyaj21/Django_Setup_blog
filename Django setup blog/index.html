Step : 1 Install Python
install python from python.org or just update.
Step : 2 Open CMD
open command promt and go to your desktop.
cd desktop
Step : 3 Create virtual env
Run the command to create virtual env. Give a name to your virtual environment inthis case env is given.And activate env.
pip install virtualenv
virtualenv env
env\scripts\activate
Step : 4 Install Django
pip install django
Step : 5 Create project
Create a django project.
django-admin startproject your_projectName_here
go inside of your project.
cd your_projectName_here
Step : 6 Open your project in live server
python manage.py runserver
Open http://127.0.0.1:8000/ in your browser
Step : 7 Move env to project
Drag env folder and drop on the project folder.
Step : 8 Open code editor
open file from desktop select your project.Open terminal and if env is not activated activate it.
Step : 9 Create app
Create a app in your project. Run command in your terminal from now. and include it in settings.py file installed apps.
python manage.py startapp app_name
